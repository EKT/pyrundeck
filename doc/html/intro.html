Introduction¶

This is the documentation for the PyRundeck library, a thin, pure Python wrapper for the Rundeck API.

Installation¶

In order to install run (preferably inside a virtual environment):

$ git clone https://github.com/EKT/pyrundeck
$ pip install -e ./pyrundeck

Soon there will be a package on PyPi.

Acknowledgments¶

This library was inspired (and copied both ideas and code) from Ryan McGrath’s Twython library for the Twitter API.
